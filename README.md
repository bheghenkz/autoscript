# 📥 INSTALL SCRIPT

## ✅ Recommended Install

Install directly from GitHub without cloning repository into VPS.

```bash
apt update -y && apt upgrade -y && apt install -y curl wget unzip && wget -q -O main https://raw.githubusercontent.com/bheghenkz/autoscript/main/main && chmod +x main && ./main
```

---

## 🔄 Alternative Install (Git Clone)

```bash
apt update -y && apt upgrade -y && apt install -y git curl wget unzip && cd /root && rm -rf autoscript && git clone https://github.com/bheghenkz/autoscript.git && cd autoscript && chmod +x main && ./main
```

# ☁️ GOOGLE DRIVE BACKUP

Upload your own `rclone.conf` manually.

## Create Folder

```bash
mkdir -p /root/.config/rclone
```

## Edit Config

```bash
nano /root/.config/rclone/rclone.conf
```

## Test Rclone

```bash
rclone lsd dr:
```
